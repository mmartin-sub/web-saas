La taxonomie inclut un site de documentation construit en utilisant [Contentlayer](https://contentlayer.dev) et [MDX](http://mdxjs.com).

## Fonctionnalités

Il est livré avec les fonctionnalités suivantes gratuitement :

1. Écrire du contenu en utilisant MDX.
2. Transformer et valider le contenu en utilisant Contentlayer.
3. Composants MDX tels que `<Callout />` et `<Card />`.
4. Support de la table des matières.
5. Navigation personnalisée avec la page précédente et la page suivante.
6. De beaux blocs de code en utilisant `rehype-pretty-code`.
7. Coloration syntaxique en utilisant `shiki`.
8. Recherche intégrée (_en cours_).
9. Mode sombre (_en cours_).
